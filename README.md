# Urban video resource management platform
平台提供视频数据的结构化处理，通过配置策略实现定制化的行业视频结构化；平台采用分布式数据存储架构，支持数据存储横向扩展，实现结构化与非结构化数据的海量存储；平台提供完备的数据存储安全策略，保障数据存储安全；平台拥有数据检索、统计与推送功能为用户提供完备的数据服务；平台开放数据服务接口为业务层的数据分析提供视频处理结果。
